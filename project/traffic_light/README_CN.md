# Traffic Light

基于 STM32F103C8T6 的红绿灯示例，仅使用 PA 引脚。

## 实现方式

- 使用封装好的 `GPIOA_MOD` 和 `GPIOA_IO_BSRR` 实现 16 位字符串点灯控制
- 所有代码目前仍在 `main.c` 中，尚未拆分为独立库文件

## 函数说明

### `GPIOA_MOD(int PAx, int mod)`

配置 PA 引脚的模式。

- `mod = 0`：浮空输入（悬空）
- `mod = 1`：推挽输出

### `GPIOA_IO_BSRR(char lights[17])`

通过 16 位字符串控制 PA0–PA15 的电平。

- `'1'`：对应引脚输出**高电平**
- `'0'`：对应引脚输出**低电平**
- 只处理前 16 个字符，第 17 位作为结束符

## 硬件连接

- PA0：红灯（低电平有效）
- PA1：黄灯（低电平有效）
- PA2：绿灯（低电平有效）
- LED 正极接 3.3V，负极接 GPIO

## 状态逻辑

红灯亮 5 秒 → 黄灯闪烁 3 次 → 绿灯亮 5 秒 → 黄灯闪烁 3 次 → 循环

## 备注

本项目用于学习 STM32 寄存器编程，不使用 HAL 库。
